# TerminalAdventure
You've lost your wallet, and have to journey deep into "The Cave of Nothing" to find it, in this randomly generated
adventure game!

# Controls
Arrow keys - Movement and selecting menu options
Enter - Open/Close item menu
X, Z  - Use items
